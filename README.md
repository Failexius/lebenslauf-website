# Mein digitaler Lebenslauf

Willkommen in meinem Repository! Hier liegt der Quellcode für meinen digitalen Lebenslauf, den ich von Grund auf neu entwickelt habe.

## Live-Ansicht

Schau dir die fertige Website direkt hier an:
**[Lebenslauf von Saidfiasal Khawarin](https://failexius.github.io/lebenslauf-website/)**

## Verwendete Technologien

- **HTML5:** Semantischer und barrierefreier Aufbau (`<header>`, `<main>`, `<aside>`, etc.)
- **CSS3:** Modernes Layout-Design mit **Flexbox**
- **Responsive Design:** Optimiert für mobile Endgeräte (Smartphones) per Media Queries
- **Font Awesome:** Einbindung von Vektor-Icons

## Über dieses Projekt

Dieses Projekt habe ich als angehender Wirtschaftsinformatiker und Junior Web Developer erstellt, um meine Kenntnisse in der modernen Frontend-Entwicklung (HTML & CSS) praxisnah anzuwenden und einen professionellen Web-Auftritt zu gestalten.
